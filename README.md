# LLM-Hub Demo Site

A static demonstration of the LLM Comparison Tool interface, showcasing the UI/UX design for comparing responses from multiple Large Language Models side-by-side.

##  Features

- **Model Comparison**: View responses from multiple AI models simultaneously
- **Dual View Modes**: 
  - Grid View: Compare 2-6 models at once
  - Carousel View: Focus on one model at a time with navigation
- **Model Selection**: Choose from GPT-3.5, GPT-4, GPT-4 Turbo, and Gemini models
- **Response Statistics**: View token count and response time for each model
- **Copy Functionality**: Easily copy model responses to clipboard
- **Responsive Design**: Optimized for desktop and mobile devices
- **Dark Mode Interface**: Modern, eye-friendly design

##  Quick Start

Simply open `index.html` in any modern web browser:

```bash
# Using a local server (recommended)
python -m http.server 8000
# Then visit http://localhost:8000

# Or double-click index.html to open directly
```

## Note

This is a **static demo** with pre-populated responses for demonstration purposes. No actual API calls are made.

For the full application with live API integration and real-time model responses, visit the [main repository](https://github.com/OscarB11/LLM-Hub-V2-beta).


##  Technology

- Pure HTML, CSS, and JavaScript
- No build process or dependencies required
- Fully client-side rendering
